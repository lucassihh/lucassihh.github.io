# Necessary Settings

## Theme

To use all the colors that the components need, it is necessary to configure the **theme.css** file.

```css
:root {
  --radius: 0.625rem;
  --background: oklch(1 0 0);
  --card: oklch(1 0 0);
  --primary: oklch(0.13 0.028 261.692);
  --secondary: oklch(0.985 0.002 247.839);
  --muted: oklch(0.556 0 0);
  --accent: oklch(0.371 0 0);
  --border: oklch(0.922 0 0);
  --input: oklch(0.922 0 0);
  --ring: oklch(0.556 0 0);
  --sidebar: oklch(0.985 0 0);

  /* COLORS 400/500/600 */
  --purple: oklch(0.714 0.203 305.504);
  --purple-accent: oklch(0.627 0.265 303.9);
  --purple-shadow: oklch(0.558 0.288 302.321);
  --yellow: oklch(0.905 0.182 98.111);
  --yellow-accent: oklch(0.852 0.199 91.936);
  --yellow-shadow: oklch(0.795 0.184 86.047);
  --green: oklch(0.792 0.209 151.711);
  --green-accent: oklch(0.723 0.219 149.579);
  --green-shadow: oklch(0.627 0.194 149.214);
  --pink: oklch(0.718 0.202 349.761);
  --pink-accent: oklch(0.656 0.241 354.308);
  --pink-shadow: oklch(0.592 0.249 0.584);
  --blue: oklch(0.707 0.165 254.624);
  --blue-accent: oklch(0.623 0.214 259.815);
  --blue-shadow: oklch(0.546 0.245 262.881);
  --orange: oklch(0.75 0.183 55.934);
  --orange-accent: oklch(0.705 0.213 47.604);
  --orange-shadow: oklch(0.646 0.222 41.116);
  --red: oklch(0.704 0.191 22.216);
  --red-accent: oklch(0.637 0.237 25.331);
  --red-shadow: oklch(0.577 0.245 27.325);
}

.dark {
  --background: oklch(0.145 0 0);
  --card: oklch(0.205 0 0);
  --primary: oklch(0.985 0.002 247.839);
  --secondary: oklch(0.13 0.028 261.692);

  --border: oklch(1 0 0 / 10%);
  --input: oklch(1 0 0 / 15%);

  --sidebar: oklch(0.205 0 0);
  color-scheme: dark;
}

  --radius-sm: calc(var(--radius) - 4px);
  --radius-md: calc(var(--radius) - 2px);
  --radius-lg: var(--radius);
  --radius-xl: calc(var(--radius) + 4px);
  --color-background: var(--background);

  --color-primary: var(--primary);
  --color-secondary: var(--secondary);
  --color-muted: var(--muted);
  --color-accent: var(--accent);
  --color-border: var(--border);
  --color-input: var(--input);
  --color-ring: var(--ring);
  --color-sidebar: var(--sidebar);

  /* Colors 400/500/600 */
  --color-purple: var(--purple);
  --color-purple-accent: var(--purple-accent);
  --color-purple-shadow: var(--purple-shadow);
  --color-yellow: var(--yellow);
  --color-yellow-accent: var(--yellow-accent);
  --color-yellow-shadow: var(--yellow-shadow);
  --color-green: var(--green);
  --color-green-accent: var(--green-accent);
  --color-green-shadow: var(--green-shadow);
  --color-pink: var(--pink);
  --color-pink-accent: var(--pink-accent);
  --color-pink-shadow: var(--pink-shadow);
  --color-blue: var(--blue);
  --color-blue-accent: var(--blue-accent);
  --color-blue-shadow: var(--blue-shadow);
  --color-orange: var(--orange);
  --color-orange-accent: var(--orange-accent);

  --color-red-shadow: var(--red-shadow);

  /* Fonts */
  --font-jakarta: "Plus Jakarta Sans", sans-serif;
}
```

## JavaScript Files

Javascript is also necessary for the functioning of some components such as `Modal`, `Accordion`, `Dropdown`, etc...

- **Download** [JavaScript Files](/src/scripts/sihh-ui-js/sihh-ui-js.zip)

After downloading, you need to import the Javascript in your `HTML`
Example for the Accordion:

```javascript
<script type="module" src="./accordion.js"></script>
```
