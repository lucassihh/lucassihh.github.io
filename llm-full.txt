# Sihh UI Full Documentation

# Sobre o template

### Clonar o repositório do github

```bash
$ git clone https://github.com/....
```

### Configurações base do template em **/src/config/pageConfig.ts**

É necessário setar a variável **BASE**, Deixe vazio para usar **/** como url base **http://localhost:4321/**

Link social para ativar o icone na **Header** em **components/Header.astro**

```ts
export const pageConfig: pageConfig = {
  SITE: "http://localhost:4321",
  BASE: "", // Url Base [!code highlight]

  SITE_NAME: "Astro-docs",
  SITE_DESCRIPTION: "Template de Documentação e Componentes",
  SITE_FAVICON: "",

  SOCIAL_LINKS: {
    FACEBOOK_URL: "",
    TWITTER_URL: "",
    GITHUB_URL: "",
    YOUTUBE_URL: "",
    DISCORD_URL: "",
  },
};
```

### Instalar dependências e iniciar o template

```bash
$ yarn
$ yarn dev
```

- Dependências principais:

```json
"dependencies": {
    "@astrojs/check": "^0.9.4",
```

# Adicionar conteúdo

### Conteúdo de documentação

- Adicionar arquivos MarkdownX **.mdx** em **/src/content/docs/nome-do-mdx**

### Configurar o Frontmatter

```
---
title: ""
description: ""
category: ""
order: 3 // Ordem na categoria.
---

```

### Links

- Todos os links relativos devem começar com **/BASE/**
- O caminho **BASE** é definido em **src/config.ts**

Não é necessário colocar a extensão **.mdx**

[Exemplo de link](/page/docs/configurar-o-layout)

`[Exemplo de Link](/page/docs/configurar-o-layout)`

--- 

### Adicionar conteúdo para o CodePreview

1. Adicionar a pasta com os arquivos HTML em **.src/components/HTML/nome-da-pasta/**
2. Após isso adicionar a pasta com os arquivos MDX em **.src/content/html-mdx/nome-da-pasta/**

### Exemplo de arquivos .mdx para os componentes HTML

1. Configurar o Frontmatter 
2. Importar o CodePreview
3. Importar os Arquivos HTML
4. Passar o arquivo HTML e o Raw para CodePreview

```markdown
---
title: "Modal"
description: "Exemplo de Modal"
category: "HTML"
order: 1
---

{/* Importar o CodePreview */}
import CodePreview from "../../../components/CodePreview.astro";

{/* Importar os Arquivos HTML */}
import Modal from "../../../components/HTML/nome-da-pasta/modal.html";
import rawModal from "../../../components/HTML/nome-da-pasta/modal.html?raw";
```

# Layout do Template

### BaseLayout

**.src/layouts/BaseLayout.astro**
Ele é o **Layout pai** e cobre os seguintes componentes do template:

- `<html>`
- `<head>`
  Carrega o componente `<BaseHead />` que contém as meta tags. e também o Open Graph
- `<body>`

### Documentation

**.src/layout/Documentation.astro** Carrega o **Header** a **Sidebar** e o **CodeCopyButton**

```astro
<BaseLayout {title} {description} {image}>
  <Header /> [!code highlight]
  <div class="flex justify-center">
    <Sidebar {id} {headings} />
      <!-- Main Content -->
      <main class="h-full w-full min-h-screen">
          <slot />
      </main>
    <CodeCopyButton />
  </div>
</BaseLayout>
```
